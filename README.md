# Bluetooth-Remote-Control-Car
The project in which the car is running using bluetooth module.

## Components used:-
>Arduino UNO
>Bluetooth Module(HC-05)
>L239D Motor Sheild
>4 DC Gear Motor
>4 Tires
>Switch
>3 Lithium Ion Battery(3.7v)
>Socket for Battery
>Jumper Wires
